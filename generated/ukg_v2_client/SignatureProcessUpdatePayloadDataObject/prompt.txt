    Description: ID of the user who created the signature process

  document_organization_ids: string[]? [nullable]
    Description: Ids of the organizations used to restrict access to the document.

For signature processes involving an employee: the employee must belong to these organizations via an active or an inactive registration references.

If this parameter is not specified, the document is associated with all of the employee’s “active” organizations for current employees or with all of the employee’s “inactive” organizations for terminated employees.

  upload_id: string? [nullable]
